# Bloom Saloon Website

Basic React.js starter project for **Bloom Saloon**, a unisex saloon in Bangalore.

## Project Purpose

- Enable clients to book appointments at Bloom Saloon (feature to be built in upcoming steps).
- Guide users to Bloom Saloon's Instagram profile.

## Current Scope

This version contains:

- A minimal React app setup.
- A simple base homepage in `src/App.js`.
- Placeholder booking button (`Book Appointment - Coming Soon`).
- Instagram redirect link.

## Tech Stack

- React.js
- react-scripts (Create React App tooling)

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

3. Open in browser:

```text
http://localhost:3000
```

## Build for Production

```bash
npm run build
```
